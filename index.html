Why ParaForm

Designed for precision.
Built for speed.

Parametric Core

Adjust sliders to customize thickness, scale, and layout. Watch meshes recalculate in real time — no manual editing.

WASM Compiler

Multi-threaded OpenSCAD runs 100% locally in your browser. Zero latency, complete privacy, full offline capability.

Manifold Export

Generate print-ready STL meshes with flawless watertight manifolds. Compatible with Bambu, Prusa, Ender, and all major slicers.

Direct SCAD Scripting

Drop into the code editor and write standard OpenSCAD. Sliders bind automatically to parameters — code and GUI stay in sync.

Workflow

From idea to print in three steps.

01

Choose a Template

Pick from a library of functional, mathematically-defined models — enclosures, mounts, tools, and more.

02

Adjust Parameters

Drag sliders or edit source code directly. The compiler re-runs instantly and updates the 3D view.

03

Export & Print

Download a watertight STL and drop it straight into your slicer. Done in minutes, not hours.

The Manifesto

We believe the 3D asset world is stuck in the era of static, uneditable meshes. ParaForm was built to transition creators to flexible, mathematical formulas.

By bringing the robust power of OpenSCAD to the browser via WebAssembly, we enable zero-overhead, highly customizable manufacturing files for everyone.

Technical Stack

  • OpenSCAD WASMMulti-threaded C++ engine, local
  • Three.js WebGLInstant interactive viewport
